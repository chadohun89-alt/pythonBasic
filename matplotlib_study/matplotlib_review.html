🖋️

한글 폰트 설정

matplotlib 기본 폰트는 한글을 지원하지 않아 깨짐이 발생한다. 사용 전 전역 설정이 필요하다.

📐

여백 & 레이아웃

x축 라벨이 길면 겹친다. rotation으로 회전, tight_layout으로 자동 여백 조정.

💾

그래프 저장

dpi 값으로 해상도를 결정한다. 72 저화질 / 150 중간 / 300 고화질.

plt.rcParams — matplotlib 전역 설정 딕셔너리.
'font.family'를 'Malgun Gothic'으로 설정하면 한글이 정상 출력된다.
'axes.unicode_minus'를 False로 설정하면 음수 기호(−)가 깨지지 않는다.
matplotlib_study1.py 
import numpy as np
import matplotlib.pyplot as plt

# ① 한글 폰트 설정 — 반드시 plot보다 먼저 설정
plt.rcParams['font.family'] = 'Malgun Gothic'
plt.rcParams['axes.unicode_minus'] = False

x = ["자바", "스프링부트", "html", "데이터베이스",
     "파이썬", "css", "javascript", "진섭이는 게임을 못한다."]
y = [45, 56, 78, 91, 68, 77, 89, 10]

plt.plot(x, y)

# ② x축 라벨 45도 회전 (길면 겹침 방지)
plt.xticks(rotation=45)

# ③ 여백 자동 설정 (라벨 잘림 방지)
plt.tight_layout()

# ④ 파일 저장 — dpi=300 고화질, transparent=True 배경 투명
plt.savefig("test2.png", dpi=300, transparent=True)

plt.show()

💡 핵심 원리

  • plt.plot(x, y) — x, y 리스트를 받아 점들을 선으로 연결한 그래프를 그린다.
  • savefig vs show — savefig()는 파일 저장, show()는 화면 출력. 순서가 중요하다. show() 이후엔 그래프가 초기화되므로 savefig를 먼저 호출해야 한다.
  • dpi (dots per inch) — 인치당 픽셀 수. 값이 클수록 선명하지만 파일 크기도 커진다.

이미지 처리 & subplot

matplotlib_study2.py
🖼️

numpy 배열 = 이미지

이미지는 결국 숫자 배열이다. 픽셀 값 0 = 검정, 255 = 흰색.

⚫

이진화 (Binarization)

픽셀을 0 또는 255 두 값으로만 나누는 처리. 경계선 감지 등에 쓰인다.

📊

subplot

하나의 창에 여러 그래프를 배치. (행, 열, 위치) 로 지정한다.

이진화 원리 — 128을 기준(threshold)으로 밝은 픽셀은 255(흰색), 어두운 픽셀은 0(검정)으로 강제 변환한다.
numpy의 조건 인덱싱을 사용하면 반복문 없이 배열 전체에 한번에 적용할 수 있다.
matplotlib_study2.py 
import numpy as np
import matplotlib.pyplot as plt

# 100×100 랜덤 픽셀 이미지 생성 (0~255 정수)
img = np.random.randint(0, 256, (100, 100))

# 원본을 건드리지 않기 위해 복사본 생성
copy_img = np.copy(img)

# ★ 이진화 — 128 이상은 255(흰색), 미만은 0(검정)으로 강제 변환
copy_img[copy_img >= 128] = 255
copy_img[copy_img < 128]  = 0

# subplot(행, 열, 위치) — 1행 2열 레이아웃 준비
plt.figure(figsize=(8, 4))

plt.subplot(1, 2, 1)          # 1행 2열의 1번째 칸
plt.imshow(copy_img, cmap='gray')
plt.title("copy")
plt.axis('off')              # 축 눈금 숨기기

plt.subplot(1, 2, 2)          # 1행 2열의 2번째 칸
plt.imshow(copy_img, cmap='gray')
plt.title("original")
plt.axis('off')

plt.show()

💡 핵심 원리

  • np.copy() — 배열을 복사한다. 그냥 =로 대입하면 같은 객체를 가리켜 원본도 바뀐다 (얕은 복사 주의).
  • 조건 인덱싱 — arr[arr >= 128] = 255 처럼 조건을 만족하는 원소만 골라 한번에 값을 바꿀 수 있다. 반복문보다 훨씬 빠르다.
  • cmap='gray' — 2D 배열을 그레이스케일로 표시. 없으면 기본 컬러맵(viridis)으로 색이 입혀진다.
  • plt.subplot(행, 열, 위치) — subplot 호출 이후의 plot 명령이 해당 칸에 그려진다.

JSON 파싱

어노테이션 데이터는 JSON 형태로 제공된다. json.load()로 딕셔너리로 변환.

바운딩 박스

객체 탐지에서 객체의 위치를 표시하는 사각형. x, y, width, height 4개 값으로 정의.

🏷️

텍스트 레이블

박스 위에 plt.text()로 클래스명 등을 표시한다.

객체 탐지 시각화 흐름
이미지 파일 + JSON 어노테이션 파일을 같이 사용한다.
JSON에는 각 객체의 카테고리 코드와 위치 좌표가 담겨 있고, patches.Rectangle로 해당 위치에 사각형을 그린다.

matplotlib_study3.py 
import matplotlib.pyplot as plt
import matplotlib.patches as patches
import json

# JSON 파일 읽기
with open('18396603_frame_20.json', 'r', encoding='utf-8') as f:
    data = json.load(f)

annotations = data['frames']['annotations']

colors = ('red', 'green', 'blue', 'yellow', 'lime',
          'magenta', 'cyan', 'purple')

# vehicle 카테고리만 좌표 수집
pos = list()
idx = 0
for ann in annotations:
    if ann['category']['code'] != 'vehicle':
        continue
    label = ann['label']
    pos.append((label['x'], label['y'],
                label['width'], label['height'],
                colors[idx]))
    idx += 1

# 이미지 출력
img = plt.imread("18396603_frame_20.png")
plt.imshow(img)

# ax = 현재 그래프 축 객체 (박스를 붙일 대상)
ax = plt.gca()

for (x, y, w, h, color) in pos:
    # 사각형 박스 객체 생성
    box = patches.Rectangle(
        (x, y),        # 좌상단 시작 좌표
        w, h,          # 너비, 높이
        fill=False,    # 내부 색 없음
        edgecolor=color,
        linewidth=2
    )
    ax.add_patch(box)  # 이미지에 박스 추가

    # 박스 위에 텍스트 표시
    plt.text(x, y - 10, 'object', color='red',
             fontsize=10,
             bbox=dict(facecolor='white', edgecolor='red', pad=2))

plt.show()

💡 핵심 원리

  • plt.gca() — get current axes. 현재 활성화된 그래프 축 객체를 반환한다. 박스나 도형을 이미지 위에 올리려면 이 축 객체가 필요하다.
  • patches.Rectangle((x,y), w, h) — (x,y)는 사각형의 왼쪽 위 모서리 좌표, 이미지 좌표계는 왼쪽 위가 원점(0,0)이다.
  • ax.add_patch(box) — 생성한 도형 객체를 실제 그래프에 등록해야 화면에 나타난다.
  • plt.text(x, y-10, ...) — y-10을 하는 이유: 박스의 위쪽 바깥에 텍스트를 배치하기 위해서 (y값이 작을수록 이미지 위쪽).

막대 그래프 & 히스토그램

matplotlib_study4.py
📊

막대 그래프 (bar)

카테고리별 수량을 비교할 때 사용. JSON을 집계해서 바로 시각화 가능.

📈

히스토그램 (hist)

연속적인 수치 데이터의 분포를 확인. 이미지 픽셀 밝기 분포 분석에 사용.

🔢

ravel()

2D 이미지 배열을 1D로 펼침. 히스토그램은 1D 배열을 입력으로 받기 때문에 필요.

matplotlib_study4.py — 막대 그래프 
import matplotlib.pyplot as plt
import json

with open("18396603_frame_20.json", "r", encoding="utf-8") as f:
    data = json.load(f)

# car, truck, person 객체 수 집계
counts = {'car': 0, 'truck': 0, 'person': 0}

for ann in data['frames']['annotations']:
    code = ann['category']['code']
    if code == 'vehicle':
        # vehicle 코드 안에 car/truck 구분이 attributes에 있음
        value = ann['category']['attributes'][0]['value']
        counts[value] += 1
    elif code == 'person':
        counts['person'] += 1

label  = list(counts.keys())    # ['car', 'truck', 'person']
values = list(counts.values())  # [갯수, 갯수, 갯수]

plt.bar(label, values, color=['blue', 'red', 'green'])
plt.ylim(0, max(values) + 1)  # y축 최대값을 데이터+1로 설정
plt.show()
matplotlib_study4.py — 히스토그램 (주석 처리된 내용) 
from PIL import Image
import numpy as np

img  = Image.open("18396603_frame_20.png")
gray = img.convert('L')      # RGB → 그레이스케일 변환

gray_arr = np.array(gray)    # PIL 이미지 → numpy 배열

plt.figure(figsize=(10, 4))

# ravel() : 2D 배열(100×100) → 1D 배열(10000,)로 펼치기
# bins=256 : 0~255 범위를 256칸으로 나눠 세밀한 분포 확인
plt.hist(gray_arr.ravel(), bins=256, color='gray')

plt.title('Histogram')
plt.xlim([0, 256])
plt.show()

💡 핵심 원리

  • 딕셔너리 집계 패턴 — counts = {'car': 0, ...} 으로 초기화 후 반복문에서 counts[key] += 1. 데이터를 집계할 때 매우 자주 쓰이는 패턴이다.
  • 히스토그램 vs 막대 그래프 — 막대 그래프는 카테고리 비교, 히스토그램은 연속 수치의 분포를 확인한다. 이미지 픽셀 분석엔 히스토그램이 적합하다.
  • img.convert('L') — Pillow에서 'L' 모드는 Luminance(밝기). RGB 3채널을 1채널 그레이스케일로 변환한다.
  • ravel() — 다차원 배열을 1차원으로 펼치는 함수. flatten()과 비슷하지만 가능하면 복사 없이 view를 반환해 더 빠르다.

실습 문제 — 크기 기준 객체 필터링

matplotlib_test.py
문제 — 125번 이미지에서 차량 중 가장 큰 차는 빨간 박스, 세 번째로 큰 차는 노란 박스로 표시하라.
핵심 포인트: 면적 = width × height로 크기를 정의하고, 가장 큰 3개를 어떻게 추출할 것인가?

🤔 풀이 전략 — 어떻게 1위, 3위를 뽑을까?

  • 변수 3개(max, max2, max3)를 두고 한 번의 순회로 상위 3개 면적 값을 추출한다.
  • 두 번째 순회에서 면적이 1위 또는 3위인 객체를 리스트에 수집한다.
  • 수집된 리스트에서 각각 좌표를 꺼내 박스를 그린다.
matplotlib_test.py 
import matplotlib.pyplot as plt
import matplotlib.patches as patches
import json

img = plt.imread('18396708_frame_125.png')

with open("18396708_frame_125.json", 'r', encoding='utf-8') as f:
    data = json.load(f)

plt.imshow(img)
ax = plt.gca()

# ① 1순위, 2순위, 3순위 면적 값 추출
max = 0;  max2 = 0;  max3 = 0

for ann in data['frames']['annotations']:
    area = ann['label']['width'] * ann['label']['height']
    if area > max:
        max = area
    elif max > area > max2:
        max2 = area
    elif max2 > area > max3:
        max3 = area

# ② 1위 또는 3위 면적을 가진 객체 수집
max_find = list()
for ann in data['frames']['annotations']:
    area = ann['label']['width'] * ann['label']['height']
    if area == max or area == max3:
        max_find.append(ann)

# ③ 박스 그리기
for i, ann in enumerate(max_find):
    lbl   = ann['label']
    color = 'red' if i == 0 else 'yellow'
    box   = patches.Rectangle(
        (lbl['x'], lbl['y']),
        lbl['width'], lbl['height'],
        fill=False, edgecolor=color, linewidth=3
    )
    ax.add_patch(box)

plt.show()

💡 핵심 원리

  • 면적 기준 정렬 vs 순회 — sorted()를 쓰면 더 간결하게 풀 수 있다. sorted(annotations, key=lambda a: a['label']['width'] * a['label']['height'], reverse=True) 로 정렬 후 [0], [2] 인덱스를 쓰면 된다.
  • elif 순서가 중요 — 면적이 max보다 크면 max를 갱신해야 하므로 큰 값부터 체크해야 한다. 순서가 바뀌면 1위·2위 판단이 틀린다.
  • == 비교의 위험 — 면적 값이 같은 객체가 여럿일 경우 오동작할 수 있다. 실무에선 정렬 후 인덱스 접근이 더 안전하다.
