① 파일 읽기

1이미지 읽기 — cv2.imread()

img = cv2.imread("opencv_study/images/coin.png")
인자타입설명
filename str 읽어올 이미지 파일 경로
flags 선택 int cv2.IMREAD_COLOR — 기본값, BGR 3채널
cv2.IMREAD_GRAYSCALE — 흑백 1채널
cv2.IMREAD_UNCHANGED — 알파 포함 4채널
반환값 → NumPy 배열 (shape: 세로 × 가로 × 채널)
OpenCV는 BGR 순서 사용 (일반 RGB와 반대)
② 가공

▲이미지 처리 흐름

1크기변환 — cv2.resize()

small = cv2.resize(img, (540, 960))
인자설명
src원본 이미지 배열
dsize(가로, 세로) 픽셀 튜플 — shape의 반대 순서 주의!
fx, fy 선택배율로 지정할 때 사용. 0.5 = 절반

2반전 — cv2.flip()

flip = cv2.flip(img, 1)
인자설명
src원본 이미지 배열
flipCode 1 — 좌우 반전 (수직 축 기준)
0 — 상하 반전 (수평 축 기준)
-1 — 상하좌우 동시 반전

3색상 처리

3-1흑백변환 — cv2.cvtColor()

gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
인자설명
src원본 이미지
code COLOR_BGR2GRAY — 컬러→흑백
COLOR_BGR2RGB — BGR→RGB
COLOR_BGR2HSV — BGR→HSV
흑백으로 바꾸면 shape가 (h, w)로 줄어든다 (채널 없어짐)

3-2경계1 — cv2.threshold()

_, thresh = cv2.threshold(blur, 131, 255, cv2.THRESH_BINARY_INV)
인자설명
src흑백(1채널) 이미지
thresh기준 밝기값 (0~255)
maxval기준 초과 픽셀에 넣을 값 (보통 255)
type THRESH_BINARY — 기준 초과 → maxval, 이하 → 0
THRESH_BINARY_INV — 반전. 기준 이하 → maxval
배경이 밝고 대상이 어두울 때 INV 사용
반환값 2개 → (실제 사용된 threshold 값, 결과 이미지). 첫 번째는 보통 _로 무시

3-3경계2 — cv2.Canny()

edge = cv2.Canny(blur, 50, 150)
인자설명
image흑백 이미지 (GaussianBlur 적용 후 권장)
threshold1낮은 기준값 — 이 이하는 edge 제외
threshold2높은 기준값 — 이 이상은 확실한 edge
기준값을 낮추면 edge가 많이 검출, 높이면 진한 경계만 남는다
사용 예: 차선 인식, 문서 스캔, 얼굴 인식, 공장 부품 균열 검사

3-4외곽선 — findContours / drawContours / contourArea

① cv2.findContours()
contours, hier = cv2.findContours(thresh, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
contour = 연결된 외곽선 묶음. threshold나 Canny로 전처리한 뒤 사용한다.
인자설명
image이진화 또는 Canny 처리된 이미지
mode RETR_EXTERNAL — 가장 바깥쪽 외곽선만
RETR_LIST — 모든 contour
RETR_TREE — 모든 contour + 계층관계 저장
RETR_CCOMP — 2계층 저장 (예: 도넛 외곽 + 구멍)
method CHAIN_APPROX_SIMPLE — 필요한 좌표만 저장 (효율적)
CHAIN_APPROX_NONE — 모든 픽셀 좌표 저장 (정밀 분석)
② cv2.drawContours()
cv2.drawContours(result, contours, -1, (0, 0, 255), 2)
인자설명
image그릴 대상 이미지 (원본 copy 권장)
contoursfindContours로 얻은 외곽선 리스트
contourIdx그릴 contour 인덱스. -1이면 전체
colorBGR 색상 튜플
thickness선 두께(px). -1이면 내부 채우기
③ cv2.contourArea()
area = cv2.contourArea(cnt) if area < 2500: continue # 작은 것 제외
반환값: 해당 contour의 픽셀 면적 (float). 노이즈나 작은 객체 필터링에 활용

3-5바운딩박스 — cv2.boundingRect()

x, y, w, h = cv2.boundingRect(cnt) cv2.rectangle(result, (x, y), (x+w, y+h), (0, 0, 255), 2)
인자설명
contour단일 contour 배열
반환값: (x, y, w, h)
x, y = 좌상단 좌표 / w, h = 너비, 높이
객체를 감싸는 직사각형 영역을 구할 때 사용

4노이즈 제거 — cv2.GaussianBlur()

blur = cv2.GaussianBlur(gray, (5, 5), 0)
인자설명
src원본 이미지
ksize커널 크기 (가로, 세로) — 반드시 홀수. 클수록 더 흐릿
sigmaX가우시안 표준편차. 0이면 ksize로 자동 계산
경계 검출 전 먼저 적용하면 노이즈가 줄어 더 깔끔한 결과
③ 보여주기

3화면 출력 — cv2.imshow()

cv2.imshow("contour", result) cv2.waitKey(0) # 키 입력까지 무한 대기 cv2.destroyAllWindows()
함수설명
imshow(winname, mat) 창을 열어 이미지를 보여준다. winname이 같으면 같은 창에 표시
waitKey(delay) 0이면 키 입력까지 무한 대기. 이미지 출력 시 사용
destroyAllWindows() 열린 모든 창 닫기
imshow() 후 반드시 waitKey()를 호출해야 창이 실제로 그려진다
④ 저장

4이미지 저장 — cv2.imwrite()

cv2.imwrite("opencv_study/images/filter_result.png", thresh)
인자타입설명
filename str 저장할 파일 경로. 확장자로 포맷 결정 (.png / .jpg 등)
img ndarray 저장할 이미지 배열
params 선택 list 품질 옵션 등 추가 파라미터 (보통 생략)
반환값 → 성공 시 True, 실패 시 False

1영상 읽기 — cv2.VideoCapture()

cap = cv2.VideoCapture("opencv_study/videos/bird.mp4")
인자타입설명
filename str / int 영상 파일 경로 또는 카메라 인덱스 (0 = 웹캠)
주요 메서드
cap.get(propId) — 속성 읽기
cap.set(propId, value) — 속성 설정. CAP_PROP_POS_FRAMES로 재생 위치 이동
cap.read() — 현재 위치 프레임 읽기 → (ret, frame) 반환 후 다음으로 자동 이동
cap.release() — 영상 파일 닫기 (메모리 해제)
width = cap.get(cv2.CAP_PROP_FRAME_WIDTH) height = cap.get(cv2.CAP_PROP_FRAME_HEIGHT) fps = cap.get(cv2.CAP_PROP_FPS) total = cap.get(cv2.CAP_PROP_FRAME_COUNT)
cap.get() / cap.set() 주요 속성의미
CAP_PROP_POS_FRAMES현재 위치 (프레임 번호)
CAP_PROP_POS_MSEC현재 위치 (밀리초)
CAP_PROP_FPS초당 프레임 수
CAP_PROP_FRAME_WIDTH영상 너비
CAP_PROP_FRAME_HEIGHT영상 높이
CAP_PROP_FRAME_COUNT전체 프레임 수

3영상 재생 루프 — while + cv2.imshow()

while True: ret, frame = cap.read() if not ret: break small = cv2.resize(frame, (540, 960)) gray = cv2.cvtColor(small, cv2.COLOR_BGR2GRAY) cv2.imshow("color", small) cv2.imshow("gray", gray) w = cv2.waitKey(30) # 30ms 대기 → fps 조절 if w == 27: # ESC 키 코드 break cap.release() cv2.destroyAllWindows()
함수설명
imshow(winname, mat) 창을 열어 현재 프레임을 보여준다
waitKey(30) 30ms 대기 → 약 33fps 속도로 재생. 반환값으로 키 입력 판별
cap.release() 루프 종료 후 영상 파일 닫기 (메모리 해제)
waitKey()의 반환값이 27이면 ESC 키 → 루프 종료
이미지 출력 시에는 waitKey(0), 영상 재생 시에는 waitKey(30) 사용
④ 저장

4특정 프레임 저장 — cv2.imwrite()

# 5초 지점 프레임으로 이동 후 저장 fps = cap.get(cv2.CAP_PROP_FPS) target_frame = int(fps * 5) cap.set(cv2.CAP_PROP_POS_FRAMES, target_frame) ret, frame = cap.read() cv2.imwrite("opencv_study/images/bird2.png", frame)
인자타입설명
filename str 저장할 파일 경로. 확장자로 포맷 결정 (.png / .jpg 등)
img ndarray 저장할 프레임 배열 (cap.read()로 읽은 frame)
params 선택 list 품질 옵션 등 추가 파라미터 (보통 생략)
반환값 → 성공 시 True, 실패 시 False
cap.set()으로 원하는 위치로 이동 → cap.read()로 해당 프레임 읽기 → imwrite()로 저장
